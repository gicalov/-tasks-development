Hello! This is my first NPM library.

# Usage

### Import:

```jsx
import { SowflakesBlock } from "falling-snowflakes-2";
```

### Settings object:

```jsx
const snowflakeOptions = {
  snowflakesCount: 50,
  snowflakeSize: 50,
  branchesCount: 6,
  rotationSpeed: 0.7,
  fallingSpeed: 2.7,
};
```

### Component display:

```jsx
<SowflakesBlock snowflakeOptions={snowflakeOptions}
```
